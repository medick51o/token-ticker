# Token Burn Ticker

**[Live ticker →](https://medick51o.github.io/token-ticker/)**

A single-file, zero-network HTML ticker showing real, measured AI token usage —
parsed from local CLI session logs by a deduplicating collector. The live headline
always has the current count. Cache reads included and labeled: every one is a real
token a model processed. Headline totals are measured only — the one estimated figure
on the page (Antigravity transcripts) is labeled as an estimate and is never summed
into the headline.

## The actual point

The ticker is simple on purpose. The **build** is the showcase: a four-vendor AI crew
(Anthropic · xAI · OpenAI · Google) orchestrated under written methods — all public as
of 2026-07-22:

- [**Anderson Method**](https://github.com/medick51o/anderson-method) — human↔model
  intent: the house rules that make the model build what you MEAN.
- [**Team Rocket Method (TRM)**](https://github.com/medick51o/team-rocket-method) —
  the engine and the crew *(the method used here)*: adversarial cross-vendor review,
  no model approves its own work, and a human is the only merge authority.
- [**TRM: Team Rocket Takes Over (TRTO)**](https://github.com/medick51o/team-rocket-takes-over) —
  the show tier: same law underneath, plus the cast and the talking cat.
- [**Anderson's Dispatch Deck (ADD)**](https://github.com/medick51o/andersons-dispatch-deck) —
  the same engine, straight-faced: no theater, plain model names.

*(This ticker was built on the method's earlier private edition — same law, now public.)*

Was a four-vendor crew overkill for a token counter? Completely — on purpose.
